# Icons Folder

This folder contains custom icons and logos used throughout the application.

## MCP Icon

To add the MCP icon, place your icon file as:
- `mcp-icon.svg` (recommended)
- Or `mcp-icon.png`, `mcp-icon.jpg`, etc.

The component will automatically use the icon if it exists, otherwise it will fallback to the default Code icon.

Supported formats: SVG, PNG, JPG, WEBP
